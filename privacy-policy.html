개인정보처리방침

최종 업데이트: 2025년 5월

Discord DM Exporter (이하 "본 확장 프로그램")는 사용자의 개인정보를 중요하게 여깁니다. 본 방침은 확장 프로그램이 수집하거나 처리하는 정보의 범위와 그 사용 방법을 설명합니다.

1. 수집하는 정보

본 확장 프로그램은 다음 정보에 접근합니다:

  • Discord 인증 토큰: Discord 웹 페이지의 로컬 스토리지에서 읽어오며, Discord API 호출에만 사용됩니다.
  • Discord 메시지 데이터: 사용자가 지정한 DM 채널의 메시지를 Discord 공식 API를 통해 가져옵니다.

2. 정보의 사용 목적

  • 인증 토큰은 Discord API에 요청을 보내기 위해 메모리에서만 사용되며, 어떠한 저장소에도 기록되지 않습니다.
  • 메시지 데이터는 사용자의 로컬 기기에 파일로 저장됩니다.

3. 외부 전송 없음

본 확장 프로그램은 수집한 어떠한 데이터도 외부 서버, 데이터베이스, 또는 제3자에게 전송하지 않습니다. 모든 처리는 사용자의 기기 내에서 완결됩니다.

4. 제3자 서비스

본 확장 프로그램은 Discord의 공식 API(discord.com/api)에만 요청을 보냅니다. Discord의 개인정보처리방침은 discord.com/privacy에서 확인하세요.

5. 데이터 보관

본 확장 프로그램 자체는 어떠한 사용자 데이터도 보관하지 않습니다. 내보내기된 파일은 사용자의 로컬 기기에 저장되며, 삭제 및 관리는 사용자의 책임입니다.

6. 아동 개인정보

본 확장 프로그램은 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 업데이트됩니다.

Privacy Policy (English)

Last updated: May 2025

Discord DM Exporter ("the Extension") respects your privacy. This policy explains what information the Extension accesses and how it is used.

1. Information Accessed

  • Discord authentication token: Read from the Discord web page's local storage, used solely to make Discord API requests.
  • Discord message data: Messages from the DM channel specified by the user, fetched via the official Discord API.

2. Use of Information

  • The authentication token is used in memory only and is never stored anywhere.
  • Message data is saved as a file on the user's local device.

3. No External Transmission

The Extension does not transmit any data to external servers, databases, or third parties. All processing occurs entirely on the user's device.

4. Third-Party Services

The Extension only makes requests to Discord's official API (discord.com/api). See Discord's privacy policy at discord.com/privacy.

5. Contact

For privacy-related questions, contact: qdyyyy@gmail.com
