# Discord DM Exporter

Discord 웹에서 특정 상대방과의 DM 전체 대화를 내보내는 크롬 확장 프로그램입니다.

## 기능

- **전체 대화 내보내기** — 페이지네이션을 자동으로 처리해 모든 메시지를 가져옵니다
- **4가지 형식 지원** — HTML / JSON / CSV / TXT
- **토큰 자동 추출** — Discord에 로그인된 세션에서 자동으로 인증 정보를 가져옵니다
- **이미지·첨부파일 링크 보존** — HTML 결과물에 이미지 미리보기 포함
- **메시지 검색** — 내보낸 HTML 파일 내 실시간 검색 지원

## 설치

### Chrome Web Store (권장)
> 스토어 링크 추가 예정

### 직접 로드 (개발자 모드)
1. 이 저장소를 클론하거나 ZIP으로 다운로드
2. `chrome://extensions` 접속
3. 우측 상단 **개발자 모드** 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 프로젝트 폴더 선택

## 사용법

1. [Discord 웹](https://discord.com) 접속 및 로그인
2. 툴바에서 **Discord DM Exporter** 아이콘 클릭
3. 상대방 **유저명** 또는 **유저 ID** 입력
   - 유저명: Discord 프로필에서 확인 (`username` 형식)
   - 유저 ID: 상대방 프로필 우클릭 → **사용자 ID 복사** (개발자 모드 필요)
4. 내보낼 형식 선택 (HTML / JSON / CSV / TXT)
5. **내보내기** 버튼 클릭

내보내기가 완료되면 브라우저 기본 다운로드 폴더에 파일이 저장됩니다.

## 출력 형식

| 형식 | 설명 |
|------|------|
| HTML | Discord UI와 유사한 스타일, 이미지 미리보기 및 검색 지원 |
| JSON | 원본 데이터 그대로 저장, 프로그래밍적 활용에 적합 |
| CSV  | 엑셀·스프레드시트에서 열기 가능 |
| TXT  | 단순 텍스트, 가독성 중심 |

## 개인정보 및 보안

- 추출한 데이터는 **로컬에만 저장**되며 외부 서버로 전송되지 않습니다
- Discord 인증 토큰은 메모리에서만 사용되며 저장되지 않습니다
- 본인 계정의 DM만 내보낼 수 있습니다

> **주의:** Discord의 이용약관에 따라 서드파티 자동화 도구 사용은 계정 정책에 위배될 수 있습니다. 본인 데이터 백업 용도로만 사용하세요.
