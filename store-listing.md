## 상세 설명

### 한국어

**Discord DM 대화 전체를 간편하게 내보내세요.**

Discord 공식 데이터 내보내기는 몇 주가 걸리고 형식도 불편합니다.
이 확장 프로그램은 원하는 상대방의 DM을 즉시, 원하는 형식으로 내보낼 수 있습니다.

**주요 기능**

✅ 대화 전체 자동 수집 — 수천 개의 메시지도 페이지 넘김 없이 자동 처리
✅ 4가지 내보내기 형식 — HTML / JSON / CSV / TXT 동시 지원
✅ Discord 스타일 HTML — 이미지 미리보기, 날짜 구분선, 메시지 검색 포함
✅ 토큰 자동 인식 — Discord에 로그인된 상태라면 별도 설정 불필요
✅ 완전 로컬 처리 — 외부 서버 전송 없음, 모든 데이터는 내 기기에만 저장

**사용 방법**

1. Discord 웹(discord.com)에 로그인
2. 툴바에서 확장 프로그램 아이콘 클릭
3. 상대방 유저명 또는 유저 ID 입력
4. 형식 선택 후 내보내기

**개인정보 보호**

수집한 데이터는 외부로 전송되지 않으며, 모든 처리는 사용자 기기 내에서 완결됩니다.

### English

**Export your entire Discord DM history — instantly.**

Discord's official data export takes weeks and comes in an inconvenient format.
This extension lets you export any DM conversation right now, in the format you want.

**Features**

✅ Full conversation export — Automatically handles pagination for thousands of messages
✅ 4 export formats — HTML, JSON, CSV, and TXT
✅ Discord-style HTML — Includes image previews, date separators, and in-page search
✅ Auto token detection — No manual setup needed if you're already logged into Discord
✅ 100% local processing — No data is sent to any external server

**How to use**

1. Open Discord Web (discord.com) and log in
2. Click the extension icon in the toolbar
3. Enter the other person's username or user ID
4. Choose your format(s) and click Export

**Privacy**

All data stays on your device. Nothing is transmitted to external servers.
